# Login League of Legends

## Descrição do projeto

Este projeto consiste em uma tela de login inspirada no jogo **League of Legends**.  
O sistema é **100% offline**, com validação local de login simulada, fundo animado (slideshow).
O objetivo principal é aplicar conceitos de **Interação Humano-Computador (IHC)**, com foco em **usabilidade** e **experiência do usuário**.

---

## Análise das metas de usabilidade

1. **Fácil de lembrar**: o layout é simples, os campos de usuário e senha estão claramente identificados e o botão de login é visível.  
2. **Fácil de entender**: feedbacks claros indicam sucesso ou erro no login, e o link “Esqueci minha senha” é óbvio para recuperação.  
3. **Útil**: cumpre a função de simular um sistema de login de forma prática e didática.  
4. **Seguro (percepção do usuário)**: o estilo visual inspirado no LoL transmite confiabilidade e familiaridade.  
5. **Eficiente**: login rápido com poucos cliques, sem passos desnecessários.

---

## Análise das metas de experiência

1. **Divertido**: o fundo animado com imagens do LoL mantém a interação interessante.  
2. **Esteticamente apreciável**: cores, fontes e layout harmonizam com a identidade visual do jogo.  
3. **Motivador**: design imersivo incentiva o usuário a interagir com o sistema.  
4. **Interessante**: slideshow no fundo chama atenção e dá sensação de dinamismo.  
5. **Satisfatório**: respostas imediatas e claras aumentam a sensação de controle do usuário.

---

## Tecnologias utilizadas

- **HTML**: estrutura da interface de login.  
- **CSS**: estilo visual, layout, efeitos de transição e animação do fundo.  
- **JavaScript**: lógica do login simulado, interação com o link “Esqueci minha senha” e slideshow do fundo.

Todas as funcionalidades são **offline**
